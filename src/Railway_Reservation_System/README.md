# Railway Reservation System

A console-based Railway Reservation System developed in Java. It allows users to register, log in, search trains, book tickets, cancel tickets, and manage reservations using file handling.

## Features

- User Registration
- User Login
- Search Trains
- Book Ticket
- Cancel Ticket
- View Booking Details
- PNR Generation
- Data stored using File Handling
- Input Validation
- Custom Exception Handling

## Technologies Used

- Java
- Object-Oriented Programming
- Collections
- File Handling
- Exception Handling

## Workflow

1. Register User
2. Login
3. Search Available Trains
4. Book Ticket
5. Generate PNR
6. Store Booking Details
7. Cancel Ticket (if required)
