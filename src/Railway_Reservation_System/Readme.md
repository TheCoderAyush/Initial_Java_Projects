# 🚆 Railway Reservation System

A console-based Railway Reservation System developed in Java that simulates the basic functionalities of an online railway booking system. The project allows users to register, log in, book train tickets, cancel bookings, and store user and passenger data using text files.

This project was built to strengthen my understanding of Java programming concepts such as Object-Oriented Programming, File Handling, Exception Handling, Collections, and modular class design.

---

## ✨ Features

- 👤 User Registration
- 🔐 User Login Authentication
- 🎫 Book Train Tickets
- ❌ Cancel Booked Tickets
- 📄 Store User and Passenger Details in Files
- 🚆 Display Available Trains
- 🆔 Generate Unique PNR Numbers
- ✅ Input Validation and Exception Handling
